# pdf-merge-test

Test merging of PDFs into one document using [pdf-lib](https://www.npmjs.com/package/pdf-lib).

### How to use

0. Clone this repo.
1. `cd pdf-merge-test`
2. `yarn`
3. `mkdir assets & mkdir out`
4. Move your PDFs into `./assets`.
5. `yarn start`
6. View your PDF at `./out/combined.pdf`.
